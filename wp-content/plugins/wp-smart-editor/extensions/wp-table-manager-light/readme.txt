More information available here: https://www.joomunited.com/wordpress-products/wp-table-manager


== Installation ==

1. Upload WP Smart Editor` to the `/wp-content/plugins/` directory or browse and upload zip file from WordPress standard installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the "Table Manager" menu to edit tables or edit it from your editor using a dedicated button

== Frequently asked questions ==

= How do I add tables in my WordPress content? =

Just open you content and you'll a button named "WP Table Manager" at the top left of your editor.
